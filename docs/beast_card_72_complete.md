# 72张异兽牌完整设计 · Beast Card Complete List
### 天地万象 · Wanxiang — Phase 1 核心内容
### 版本 V1.0 · 2026.02.19

---

## 设计原则

1. **每张异兽有名有典**：名字出自山海经/中国神话，附典故来源
2. **效果服务于游戏性**：文化元素自然融入，不生硬
3. **四象均衡**：每象18张，稀有度分布严格遵守框架
4. **12种效果类型覆盖**：框架V2中定义的数值范围严格执行
5. **命名双语**：中文名（主要） + 英文名（代码用）

---

## 编号规则

`B-{四象}{稀有度}{序号}`
- 四象：QG=青龙, ZQ=朱雀, BH=白虎, XW=玄武
- 稀有度：C=凡兽, U=灵兽, R=神兽, L=传说
- 示例：B-QGL01 = 青龙传说第1张

## B-QGL01 · 饕餮（Tao Tie）

| 属性 | 值 |
|:---|:---|
| **中文名** | 饕餮 |
| **英文名** | Taotie, the Insatiable |
| **四象** | 青龙♠（木·生长） |
| **稀有度** | 🟡 传说（太古神兽） |
| **效果类型** | ⑩ 递增/累积型 |
| **典故** | 《山海经·北次二经》："其状如羊身人面，其目在腋下，虎齿人爪，其音如婴儿。"上古四大凶兽之一，贪食无度，吞噬一切。 |

**效果**：
```
触发: ON_SELL（卖出其他异兽时触发）
效果: 每当你卖出一张异兽牌，饕餮永久获得 +该异兽卖出价格 的 Mult
```

| 参数 | 值 |
|:---|:---|
| 触发类型 | ON_SELL（他兽被卖出时） |
| 效果类型 | SCALING_MULT |
| 初始Mult | +0 |
| 增长 | +卖价/手（永久），如卖$3灵兽=+3 Mult |
| 购买价格 | $12 |
| 卖出价格 | $6 |

**描述（中文）**：「每当你卖出一只异兽，永久获得等同其卖价的倍率加成。」
**描述（English）**：`Each time you sell a Beast, permanently gain +Mult equal to its sell price.`

**设计理念**：
- 鼓励"吞噬"——不断买卖异兽来喂养饕餮
- 前期弱（需要卖出异兽=主动放弃资源），后期极强
- 8个卦(Ante)大约有6-10次买卖机会，卖价平均$3→+18~30 Mult
- 与经济型异兽协同：先用经济型赚钱→买更多异兽→卖给饕餮
- **典故呼应**：饕餮贪吃万物 = 吞噬其他异兽转化为力量

## B-ZQL01 · 毕方（Bi Fang）

| 属性 | 值 |
|:---|:---|
| **中文名** | 毕方 |
| **英文名** | Bifang, the Fire Herald |
| **四象** | 朱雀♥（火·爆发） |
| **稀有度** | 🟡 传说（太古神兽） |
| **效果类型** | ⑦ 自毁/牺牲型 |
| **典故** | 《山海经·西山经》："有鸟焉，其状如鹤，一足，赤文青质而白喙，名曰毕方，其鸣自叫也，见则其邑有讹火。"火焰之鸟，出现之处必生大火。 |

**效果**：
```
触发: ON_SELL（自身被卖出时）
效果: 卖出毕方时，复制左侧异兽牌（创建永久副本填入空栏位）
```

| 参数 | 值 |
|:---|:---|
| 触发类型 | ON_SELL（自身） |
| 效果类型 | SELF_DESTROY_GAIN（特殊：复制） |
| 效果 | 复制左侧异兽，永久副本 |
| 购买价格 | $10 |
| 卖出价格 | $5 |

**描述（中文）**：「卖出毕方时，左侧的异兽获得一个永久副本（填入空栏位）。涅槃之火，赐予新生。」
**描述（English）**：`When sold, create a permanent copy of the Beast to the left.`

**设计理念**：
- 终极一次性效果——牺牲传说级异兽换一个任意异兽的复制
- 最佳用法：复制另一个传说级异兽（如复制饕餮=双饕餮）
- 时机选择极其重要：太早卖没有好异兽可复制，太晚卖可能栏位满
- **典故呼应**：毕方=火之神鸟→涅槃→从火焰中重生=复制新生

| **稀有度** | 🟡 传说（太古神兽） |

**效果**：
```
触发: ON_SCORE
效果: 所有计分牌重触发1次（每张计分牌的筹码贡献和增强效果再算一遍）
```

| 参数 | 值 |
|:---|:---|
| 触发类型 | ON_SCORE |
| 效果类型 | RETRIGGER（value=1，target=ALL） |
| 影响范围 | 所有计分牌 |
| 购买价格 | $12 |
| 卖出价格 | $6 |

**描述（中文）**：「所有计分牌额外触发一次。龟承天地，万物重来。」
**描述（English）**：`All scoring cards are retriggered once.`

**设计理念**：
- 最强的后期放大器——所有计分牌的筹码值和增强效果翻倍
- 裸牌时效果温和（5张×10 Chips = 额外+50 Chips）
- 配合多彩增强（×1.5 Mult/张）→ 5张全多彩+玄龟=5次额外×1.5=×7.59
- **终极Build**：多彩牌组+玄龟=指数级爆发
- **典故呼应**：龟=永恒、重复、承载=重触发=一切再来一次
