# 16卷天书完整设计 · Celestial Tome Complete List
### 天地万象 · Wanxiang — Phase 1 核心内容
### 版本 V1.0 · 2026.02.20

---

## 设计原则

1. **16 = 8 × 2（八卦 × 两仪）**：8卷基础天书 + 8卷进阶天书，严格配对
2. **基础→进阶**：领悟基础天书后，对应进阶天书才会出现在商店
3. **永久被动增益**：天书一经领悟，永久生效直到本局结束
4. **道经为源**：每卷天书以道教经典/易经卦辞命名，文化纯度100%
5. **功能覆盖**：完全覆盖Balatro全部Voucher功能 + 天地万象原创功能

---

## 编号规则

`T-{B/A}{序号}`
- B = Base（基础），A = Advanced（进阶）
- 基础B01~B08，进阶A01~A08
- 基础Bn 解锁后 → 进阶An 才出现在商店

---

## 核心机制

### 获取方式
- 每个卦（Ante）的商店中出现 **1卷天书**（随机）
- 价格统一 **$10**
- 每卦只能购买1卷
- 基础天书领悟后，对应进阶天书加入候选池

### 进阶解锁链
```
基础天书 B01 领悟 → 进阶天书 A01 解锁（可出现在后续商店）
基础天书 B02 领悟 → 进阶天书 A02 解锁
... 依此类推 ...
```

### TAB面板显示
- 状态面板中显示已领悟的天书列表
- 基础天书显示为铜色卷轴图标
- 进阶天书显示为金色卷轴图标

道德经是天地万象的**核心天书**，领悟后解锁6张出牌 → 解锁11种新牌型 → 解锁11张稀有星宿。这是整个游戏Build的战略性分水岭。配对路径：阴符经(卖价+$2) → 道德经(6张出牌)，意味着玩家需要先学会"以退为进"(经济管理)，才能领悟"道"(突破限制)。
