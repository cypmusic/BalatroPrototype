# 二、核心数理体系

所有数字均源自中国传统数理哲学，互相之间存在深层关联：

```
太极
 ├→ 两仪 (2)
 ├→ 四象 (4) ← 灵印4种
 ├→ 八卦 (8)
 ├→ 十殿阎罗 (10) ← 幽冥牌10张 【V2新增暗线】
 ├→ 十六天书 (16) = 8 × 2
 ├→ 二十八星宿 (28) = 4 × 7（完全数）
 ├→ 三十六天罡 · 法宝 (36) = 法宝16 + 阵法10 + 幽冥10
 ├→ 五十二神将 (52)
 ├→ 六十四卦 · 路线 (64) = 8 × 8
 └→ 七十二地煞 · 异兽 (72) = 36 × 2
```

| 系统 | 数量 | 文化来源 | 数理关系 |
|:---|:---:|:---|:---|
| 路线系统 | **64** | 易经六十四卦 | 8 × 8 = 64 |
| 异兽牌 | **72** | 七十二地煞 | 36 × 2 = 72 |
| 神将牌 | **52** | 五十二周天 | 扑克牌基数 |
| 法宝牌 | **36** | 三十六天罡 | 64 − 28 = 36 |
| 星宿牌 | **28** | 二十八宿 | 4 × 7 = 28 |
| 天书 | **16** | 八卦 × 两仪 | 8 × 2 = 16 |

### 隐藏数理暗线

- **暗线一**：36(天罡) + 72(地煞) = **108**（天罡地煞合数）
- **暗线二**：28(星宿) + 36(法宝) = **64**（星宿与法宝合为天道）
- **暗线三**：72 = 64 + 8（地煞 = 六十四卦 + 八卦）
- **暗线四**：16 = 8 × 2（天书 = 八卦 × 两仪）
- **暗线五【V2新增】**：10(幽冥) = **十殿阎罗**（冥府十殿）
- **暗线六【V2新增】**：4(灵印) = **四象**（青龙/朱雀/白虎/玄武各一印）
- **暗线七【V2新增】**：16(法宝神器) = **十六天书**的镜像（天书=被动永久，法宝=主动一次性）

### 需新增的触发/效果类型

**新增 TriggerType**：
- `ON_ROUND_END` — 回合结束时触发（经济型）
- `ON_BLIND_SELECT` — 选择盲注时触发
- `ON_SELL` — 被卖出时触发（牺牲型）
- `ON_CARD_DESTROYED` — 牌被销毁时触发
- `ON_JOKER_ADDED` — 新异兽入队时触发
- `ALWAYS_PASSIVE` — 纯被动（如：手牌上限+1）
- `ON_REEL_DRAW` — 天机抽牌时触发 **【V2新增】**
- `ON_BEAT_HIT` — 踩中节拍时触发 **【V2新增】**

**新增 EffectType**：
- `EARN_MONEY` / `EARN_MONEY_PER` — 获得金币
- `RETRIGGER` — 重触发计分牌
- `COPY_LEFT` / `COPY_RIGHT` — 复制相邻异兽效果
- `SELF_DESTROY_GAIN` — 自毁并获得效果
- `SCALING_MULT` / `SCALING_CHIPS` — 递增倍率/筹码
- `CREATE_CONSUMABLE` — 创建消耗品
- `MODIFY_HAND_SIZE` — 改变手牌/出牌上限

| 难度递进 | 8个难度等级，每级8个卦可选 |
| 重玩价值 | 几千条有效路线 × 不同Build × 不同终局Boss |

## 6.2 单局流程

```
第1卦（起始） → 小盲注 → 大盲注 → Boss盲注 → 商店 → 选择下一卦（2-3分岔）
第2卦        → 小盲注 → 大盲注 → Boss盲注 → 商店 → 选择下一卦
...（重复）...
第8卦（终卦） → 小盲注 → 大盲注 → 最终Boss → 通关！
```

## 6.3 卦象对游戏的影响

| 卦象属性 | 游戏影响 | 举例 |
|:---|:---|:---|

## 6.4 四象Boss

- 走白虎系卦多 → 最终面对白虎Boss

## 6.5 卦象流派

玩家的牌组状态 + 走过的卦 → 实时映射到一个"当前卦象"，提供全局被动加成并影响商店和掉落。
